# Powder Keg

Powder Key is a real-time strategy game set on a 1:1 scale replica of Earth, utilizing NASA geographic data for terrain and elevation. Players command global military forces, managing ballistic missiles, aircraft, and supply lines in a high-stakes conflict. The game emphasizes realistic military strategy, resource management, and geopolitical decision-making.
